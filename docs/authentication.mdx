---
title: "Authentication"
mode: "center"
---

Authenticate your account by including your **secret API key** in every request.  
You can create and manage API keys from the integration page at <a href="https://app.getfernand.com" target="_blank">Fernand's Integrations</a>.

Send your key using the following HTTP header:

`X-API-Key: sk_xxxxxx.....`
